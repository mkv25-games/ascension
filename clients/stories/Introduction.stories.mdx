# Welcome to Ascension

Ascension is a game project that combined statically hosted HTML/JS built from component, combined together into experiences that are accessible from web enabled devices such as mobile phones, tablets, laptops, and personal computers.

The stories in this project are designed to test out the individual components independent of data used to host the service.
